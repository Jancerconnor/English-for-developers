🧠 English Quest

Aprende inglés técnico jugando. Un quiz interactivo hecho a medida para desarrolladores que necesitan dominar el inglés que se usa todos los días en el trabajo: documentación, code review, entrevistas técnicas, arquitectura y más.


Proyecto personal creado para reforzar mi inglés técnico de forma independiente, además de mis clases presenciales. En vez de usar una app genérica, construí la mía propia — con el vocabulario y las situaciones que realmente uso como programador.


🎯 ¿Qué es esto?

English Quest es un juego de preguntas y respuestas con 900 preguntas repartidas en 9 módulos, organizados por nivel del Marco Común Europeo de Referencia (CEFR), desde vocabulario básico hasta conversación técnica avanzada de arquitectura de software.

No es un curso de inglés genérico: cada pregunta está pensada para el contexto real de un desarrollador — leer documentación, escribir mensajes de commit, sobrevivir a un code review, o defenderte en una entrevista técnica en inglés.
